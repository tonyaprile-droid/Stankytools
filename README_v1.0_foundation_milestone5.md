# StankyTools v1.0 Foundation Milestone 5

## Auction House Trading Terminal

This milestone begins the premium Auction House redesign.

### Added
- Rebuilt Auction House page into a trading-terminal layout.
- Large item intelligence panel with artwork preview.
- Current/average/supply/profit stat cards.
- Lightweight price history graph.
- Watchlist and favorite toggles saved locally.
- Scanner bridge retained in the Market page.
- Table selection now drives the item intelligence panel.

## Easter Egg

- Press **Ctrl + Alt + S** inside the app to open the hidden in-app transmission popup.
- If Qt WebEngine is not available, it opens the video in the user's browser instead.

Updated to `1.0.0-foundation.5`.
