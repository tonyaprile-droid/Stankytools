# Phase 2.8 - Event Central Time

Changes:
- Event creation now defaults to Central Time.
- Removed the timezone dropdown from the Add Event dialog.
- Event creation labels the picker as `Date / Time (Central)`.
- Guild Events section now states that all event times are Central Time (CT).
- Dashboard Events section now states that event times are Central Time (CT).
- Event lists and detail dialogs now show the event time, not just the date.
