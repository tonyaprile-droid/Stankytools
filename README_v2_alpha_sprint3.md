# StankyTools v2 Alpha Sprint 3

## Included
- Uses the provided Deep Desert grid image as the local Deep Desert map.
- Dashboard now replaces Market Coverage with Guild Activity Feed.
- Dashboard now replaces Recent Market Activity with Guild News.
- Guild Operations logo area is larger and includes member count.
- New Guild page in the main menu.
- Guild page includes members/roles, guild news, and useful links.
- Owners/officers can submit Guild News.
- Owners/officers can add/edit/delete Useful Links.
- Custom guild code supported when creating a guild.
- Existing map, POI, base, catalog image, and sync improvements are preserved.

## Supabase
Run `SUPABASE_UPDATE_v1_8.sql` in Supabase SQL Editor to enable Guild News, Activity Feed, and Useful Links sync.

Paste the SQL contents into Supabase, not just the filename.
