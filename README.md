# Stanky Market - Deep Desert POI Map Update

Run with:

```powershell
py main.py
```

Changes in this build:
- Removed the Deep Desert map status section.
- Removed the extra header/top bar above the map so the map gets more room.
- Map now auto-fits to the available space.
- POI list is cleaner.
- Clicking a POI in the list jumps/centers the map to that POI.
- Clicking an existing marker on the map selects and centers it.
