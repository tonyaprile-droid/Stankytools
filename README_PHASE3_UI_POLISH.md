# Phase 3 UI Polish

This phase focuses only on final UI polish and consistency.

Included:
- More consistent Dune-themed button, menu, dialog, tooltip, table, and input styling.
- Styled danger and ghost buttons so they no longer look like default Windows controls.
- Cleaner muted labels, guild status pills, and guild logo frames.
- Better combo box/dropdown/date-time styling.
- Softer card shadows on dashboard cards, news cards, link cards, event cards, and toast notifications.
- Cleaner table row heights and header sizing for better readability.
- Small UX tooltips for event response buttons.

No database schema changes are required for this phase.
